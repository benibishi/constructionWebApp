# BuildMaster - Construction Management Web App

A comprehensive construction management web application built with HTML, CSS, and JavaScript. This app helps construction companies manage projects, tasks, and teams efficiently.

## 🏗️ Features

### ✅ Project Dashboard
- Project overview with progress tracking
- Key statistics and metrics
- Recent activity feed
- Project creation and management

### ✅ Task Management
- Create and assign tasks with priorities
- Filter tasks by project, status, and search
- Track task progress and deadlines
- Task editing and deletion capabilities

### ✅ Team & Role Management
- Team member profiles with roles
- Contact information management
- Role-based organization
- Member addition and removal

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start managing your construction projects!

### Usage
1. **Dashboard**: View project overview and key metrics
2. **Tasks**: Manage project tasks and assignments
3. **Team**: Organize team members and roles

## 🔧 Customization

### Adding New Features
1. Create new HTML sections in `index.html`
2. Add corresponding CSS in `css/` directory
3. Implement JavaScript logic in `js/` directory
4. Update navigation as needed

### Styling
- Modify `css/styles.css` for global styles
- Adjust `css/dashboard.css` for component-specific styles
- Use CSS variables for consistent theming
